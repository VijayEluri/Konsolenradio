Was kann das Konsolenradio?

Das Konsolenradio ist ursprünglich ein Shellskript zum abspielen von Audiostreams. Bei der JVM-Edition wurde die Funktion des Skriptes in der Sprache Java auf die JVM portiert. Im Verzeichnis /bin/ befindet sich eine ausführbare Jar-Datei und als "Starthilfe" eine Batch-Datei für Windows und ein Bashskript für Linux. Diese starten jedoch lediglich die Jar-Datei.

Machen sie das Skript unter Linux mit dem Befehl 'chmod +x .radio.sh' ausführbar und rufen sie es durch ./radio.sh dann auf.

Eine Liste von voreingestellten URLs finden sie unter Radiosender
